PlmIndex

       PlmIndex - Compute the index of an array corresponding to degree L and
                  angular order M.

SYNOPSIS

       INTEGER FUNCTION PlmIndex ( L, M )

           INTEGER   L, M

DESCRIPTION

       PlmIndex will calculate the index of an array corresponding to degree L
       and angular order M. The input arrays are generated by routines such as
       PlmBar, PlmBar_d1, PlmSchmidt, PlmSchmidt_d1, PlmON, PlmON_d1,
       PLegendreA, and PLegendreA_d1. The array index is equal to
       L*(L+1)/2+M+1.

ARGUMENTS

       L         (input) INTEGER

                 The spherical harmonic degree.

       M         (input) INTEGER

                 The angular order.

SEE ALSO

       plmbar(1), plmbar_d1(1), plmschmidt(1), plmschmidt_d1(1), plmon(1),
       plmon_d1(1), plegendrea(1), plegendrea_d1(1)
