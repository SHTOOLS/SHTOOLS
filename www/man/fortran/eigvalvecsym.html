EigValVecSym

       EigValVecSym -  Compute the eigenvalues and eigenvectors of a real
                       symmetric matrix.

SYNOPSIS

       SUBROUTINE EigValVecSym ( AIN, N, EVAL, EVEC, UL, K )

           REAL*8                AIN(N, N), EVAL(N) or EVAL(K), EVEC(N, N) or
                                 EVEC(N, K)

           INTEGER               N

           CHARACTER, OPTIONAL   UL

           INTEGER, OPTIONAL     K

DESCRIPTION

       EigValVecSym will calculate the eigenvalues and eigenvectors of a real
       symmetric matrix. By default, only the upper portion of the matrix is
       used, but this can be changed by the optional argument UL. The
       eigenvalues and eigenvectors are sorted from largest to smallest
       eigenvalues. If the optional parameter K is specified, then only the K
       largest eigenvalues and their corresponding eigenvectors will be
       output.

       The matrix Ain is first factorized into a tridiagonal matrix using the
       LAPACK routine DSYTRD, and then the eigenvalues are calculated by calls
       to DSTEGR and DORMTR.

ARGUMENTS

       AIN       (input) REAL*8, DIMENSION (N, N)

                 The input real symmetric matrix. By default, only the upper
                 portion of the matrix is used.

       N         (input) INTEGER

                 The rank of the matrix AIN.

       EVAL      (output) REAL*8, DIMENSION (N) or DIMENSION (K)

                 The eigenvalues of AIN, sorted from largest to smallest.

       EVEC      (output) REAL*8, DIMENSION (N, N) or DIMENSION (N, K)

                 The eigenvectors of AIN, sorted from largest to smallest
                 eigenvalues. The sign of the first element of each
                 eigenvector is chosen to be positive.

       UL        (input) CHARACTER, OPTIONAL

                 If "U" then the upper portion of the matrix AIN will be used
                 (default). If "L" then the lower portion of the matrix AIN
                 will be used.

       K         (input) INTEGER, OPTIONAL

                 The number of largest eigenvalues and corresponding
                 eigenvectors to be output.

NOTES

       This routine requires the matrix algebra packages BLAS and LAPACK which
       are available at <http://www.netlib.org/lapack/> and
       <http://www.netlib.org/blas/>, respectively.
