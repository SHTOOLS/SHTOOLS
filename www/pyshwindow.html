SHWindow

Subclasses
SHSymmetricWindow Class for windows concentrated within a spherical cap.
SHAsymmetricWindow Class for windows concentrated in an arbitrary domain.
Initialization
x = SHWindow.from_cap() Construct windows concentrated with a spherical cap.
x = SHWindow.from_mask() Construct windows concentrated with an arbitrary region.
Methods
plot() Plot the best concentrated windows.
get_spectrum() Return the power spectrum of the localization windows.
get_couplingmatrix() Return the coupling matrix of the first nwin tapers.
plot_couplingmatrix() Plot the coupling strength of the localization windows.
info() Print meta information about the tapers.
