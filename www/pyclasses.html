SHCoeffs

Subclasses
SHRealCoefficients Real spherical harmonic coefficient class.
SHComplexCoefficients Complex spherical harmonic coefficient class.
Initialization
SHCoeffs.from_array() Initialize using coefficients from an array.
SHCoeffs.from_random() Initialize using random coefficients with a given power spectrum.
SHCoeffs.from_file() Initialize using coefficients from a file.
Functions
SHCoeffs.get_degrees() Return the array [0,...,lmax] that contains the ordered degrees l.
SHCoeffs.get_powerperdegree() Return the power per degree spectrum.
SHCoeffs.get_powerperband() Return the power per logbandwidth spectrum.
SHCoeffs.get_coeffs() Return complex or real spherical harmonics coefficients in different normalizations.
SHCoeffs.rotate() Rotate the spherical harmonics coefficients by [alpha, beta, gamma].
SHCoeffs.expand() Expand the coefficients to a spherical grid. Available grid types:
kind ='DH1': equidistant lat/lon grid with nlat=nlon
kind ='DH2': equidistant lat/lon grid with 2*nlat=nlon
kind ='GLQ': Gauss Legendre Grid
SHCoeffs.plot_powerperdegree() Plot the power per degree spectrum.
SHCoeffs.plot_powerperband() Plot the power per logbandwidth(degree) spectrum.
SHCoeffs.make_complex() Convert the real coefficient class to the complex harmonic coefficient class.
SHCoeffs.make_real() Convert the complex coefficient class to the real harmonic coefficient class.

SHGrid

Subclasses
DHGrid Class for Driscoll and Healy (1994) sampled grids.
GLQGrid Class for Gauss-Legendre quadrature sampled grids.
Initialization
SHGrid.from_array() Initialize using an array.
SHGrid.from_file() Initialize using an array from a file.
Functions
SHGrid.get_lats() Return a vector containing the latitude of each row in the grid.
SHGrid.get_lons() Return a vector containing the longitude of each column in the grid.
SHGrid.plot_rawdata() Plot the grid using a two dimensional cylindrical projection.
SHGrid.expand() Expand the gridded data into spherical harmonics.

SHWindow

Subclasses
SymmetricWindow Class for windows concentrated within a spherical cap.
AsymmetricWindow Class for windows with arbitrary concentration domains.
Initialization
SHWindow.from_cap() Construct windows concentrated with a spherical cap.
SHWindow.from_mask() Construct windows concentrated with an arbitrary region.
Functions
SHWindow.plot() Plot the best concentrated windows.
SHWindow.get_spectrum() Return the power spectrum of the localization windows.
SHWindow.get_couplingmatrix() Return the coupling matrix of the first nwin tapers.
SHWindow.plot_couplingmatrix() Plot the coupling strength of the localization windows.
SHWindow.info() Print meta information about the tapers.
